# Grip-Technical

## Installation & Usage

### Technologies

- `TypeScript`
- `NodeJS`
- `Jest`
- `validator`

### Installation

- Clone or download the repo
- Open terminal and run `npm install`

### Usage

- First run `npm test` to run all tests
- run `tsc` to compile typescript code
- run `node dist/src/task2.js`
- run `node dist/src/tasks3-6.js`
